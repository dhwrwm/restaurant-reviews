# Future Ideas

<Callout type="warning">**Status:** 📋 Planned — unscoped, no committed phase.</Callout>

Ideas that are plausible extensions of the platform but aren't detailed
enough yet to belong to a specific roadmap phase:

- [Reservations](/features/reservations) — the least-scoped feature on the
  board; booking model and availability logic are both undecided.
- Multi-language / i18n support.
- Owner-facing analytics dashboard (distinct from the backend
  [Analytics](/features/analytics) service — this would be the UI on top of
  it).
- Admin role / moderation tooling (today there are only `REVIEWER` and
  `OWNER` — see [Features: Authorization](/features/authorization)).

Items move out of this page and into a numbered phase once they're scoped
enough to have a concrete planned shape, following the pattern on the other
[Roadmap](/roadmap) pages.
