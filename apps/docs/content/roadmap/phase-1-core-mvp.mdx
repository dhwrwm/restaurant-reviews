# Phase 1 — Core MVP

<Callout type="success">**Status:** ✅ Complete</Callout>

- [Authentication](/features/authentication) — JWT access/refresh, Argon2,
  rotation.
- [Authorization](/features/authorization) — `REVIEWER`/`OWNER` roles,
  ownership checks.
- [Restaurants](/features/restaurants) — owner CRUD, public filterable
  listing.
- [Reviews](/features/reviews) — reviewer CRUD, one review per reviewer per
  restaurant.
- Base [Database](/database) schema and [API](/api) conventions.

This is the system described in full on
[Current Architecture](/architecture/current) — nothing in this phase is
aspirational.
