# API Overview

<Callout type="info">**Status:** ✅ Implemented</Callout>

The backend exposes a REST API under a global `/api` prefix (e.g.
`https://<service>.onrender.com/api`). There is no GraphQL or RPC layer.

## Global conventions

- **Validation** — every request body is validated with a global
  `ValidationPipe` (`whitelist`, `forbidNonWhitelisted`, `transform`):
  unknown fields are rejected, not silently dropped.
- **Auth** — JWT access tokens as HTTP-only cookies, checked by
  `JwtAuthGuard`; see [API: Authentication](/api/authentication).
- **CORS** — restricted to a single configured origin (`WEB_URL`), with
  credentials enabled.
- **Security headers** — `helmet()` on every response.
- **Rate limiting** — a global limit, plus a stricter limit on
  `/auth/register` and `/auth/login`.
