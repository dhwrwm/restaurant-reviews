# Restaurant Review Platform

Engineering documentation for the **Restaurant Review Platform** — a
full-stack application for discovering, rating, and reviewing restaurants.

## Technology Stack

- **Web** — Next.js, React, Tailwind CSS
- **API** — NestJS, Prisma
- **Database** — PostgreSQL
- **Monorepo** — Turborepo, pnpm workspaces
- **Docs** — Next.js, Nextra
